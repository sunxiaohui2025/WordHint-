# 🎯 GOAL：开发 Chrome 英语学习翻译插件「WordHint」

## 一、产品目标
开发一个 Chrome 浏览器翻译插件，类似于沉浸式翻译插件。但它**不做全文翻译**，只挑出网页中的**疑难英文单词**，把中文释义**紧贴标注在单词正上方**（ruby 振假名样式），帮助用户在阅读英文原文时降低障碍、同时学习词汇。

## 二、核心功能需求

### 1. 疑难词判定（两步走）
- **第一步·本地词表筛选**：对页面英文分词，与内置词表比对，筛出候选疑难词。
- **难度档位**（用户可在插件面板切换）：
  - 档 1「四级」：四级词表以外的词判为疑难
  - 档 2「六级」：六级词表以外的词判为疑难
  - 档 3「进阶」：六级以外，聚焦雅思/托福等更高难度词
- **过滤白名单**：候选词先剔除白名单中"我认识"的词。
- **第二步·大模型释义**：将候选疑难词**连同所在句子**批量发给本地大模型，获取该词**在当前句子中的**中文含义。

### 2. 标注展示
- 用 HTML `<ruby>` 标签，将中文释义以小字标注在英文单词**正上方**。
- **悬停/点击**单词时，弹出浮层展示**更详细的解释**（如：词性、完整释义、例句）。
- 浮层内提供两个操作按钮：**「我认识」→ 加入白名单**、**「收藏」→ 加入学习名单**。

### 3. 生词本（本地存储）
- **白名单**：标记为"认识"的词，后续页面**不再标注**。
- **学习名单**：收藏的词，供日后复习。
- 两个名单均可在插件面板中**查看、增删**。
- 学习名单支持**导出 CSV**，字段：`单词，中文释义，来源句子，收藏时间`。

## 三、技术约束

| 项 | 规格 |
|---|---|
| 目标浏览器 | 仅 Chrome（Manifest V3） |
| 词表来源 | 使用公开的四级/六级/雅思托福词表（txt），内置于插件 |
| 大模型接口 | vLLM，OpenAI 兼容格式 |
| BASE_URL | 见 `.env` 文件（本地配置） |
| MODEL | 见 `.env` 文件（本地配置） |
| API_KEY | 见 `.env` 文件（本地配置） |
| **关键参数** | 请求体必须带 `"chat_template_kwargs": {"enable_thinking": false}` 以关闭思维链，否则会超时/浪费 token |
| 释义请求 | 批量提交、结合句子上下文、`temperature: 0`，返回 JSON 数组 `[{"word":"","meaning":""}]` |
| 数据存储 | 全部存 `chrome.storage.local`，不上传任何服务器 |

## 四、已验证的接口调用范例（务必遵循）

配置信息存储在 `.env` 文件，以下为调用结构范例：

```bash
POST {BASE_URL}/{MODEL}/v1/chat/completions
Authorization: Bearer {API_KEY}
{
  "model": "{MODEL}",
  "messages": [
    {"role":"system","content":"你是英语词汇助手。根据句子上下文，为给定的英文难词给出它在该句中的中文含义。只输出 JSON 数组，格式 [{\"word\":\",\"meaning\":\"}]，meaning 为简短中文，不要解释。"},
    {"role":"user","content":"句子：...\n难词：word1, word2, word3"}
  ],
  "temperature": 0,
  "max_tokens": 300,
  "chat_template_kwargs": {"enable_thinking": false}
}
# 已验证返回：[{"word":"leverage","meaning":"利用"},{"word":"proprietary","meaning":"专有的"},{"word":"mitigate","meaning":"减轻"}]
```

## 五、验收标准（Definition of Done）
1. ☐ 插件能在 Chrome 以开发者模式加载（Manifest V3，无报错）。
2. ☐ 打开任一英文网页，页面中的疑难词上方出现中文小字标注。
3. ☐ 切换难度档（四级/六级/进阶），被标注的词数量随之变化（档位越高标注越少）。
4. ☐ 悬停/点击标注词，弹出含详细解释的浮层，并有「我认识」「收藏」按钮。
5. ☐ 点「我认识」后刷新页面，该词不再被标注。
6. ☐ 点「收藏」后，该词出现在学习名单中。
7. ☐ 学习名单可导出为 CSV，字段包含：单词、释义、来源句子、时间。
8. ☐ 白名单/学习名单数据在浏览器重启后仍然保留（本地持久化）。
9. ☐ 大模型请求已关闭 thinking，单页翻译响应正常、无长时间卡顿。
10. ☐ 全程无任何数据上传到第三方服务器。
